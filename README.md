# Multi-Page College Club Website

This repository contains a multi-page College Club website created as part of the **Multi-Page College Club** web development assignment. The project is built using only **HTML** and **basic CSS**, and demonstrates semantic structure, layout techniques, and HTML forms. [file:1]

## Project Structure

The website consists of the following pages: [file:1]

- `index.html` – Home page  
- `about.html` – About Club page  
- `events.html` – Events / Activities page  
- `join.html` – Join Club form page  
- `styles.css` – Common stylesheet  
- `images/club.jpg` – Club-related image used on the About page  

## Features Implemented

- **Semantic HTML layout** using `<header>`, `<nav>`, `<main>`, and `<footer>` where appropriate. [file:1]  
- **Common Navigation Bar** on all pages with links to:
  - Home
  - About
  - Events
  - Join [file:1]
- **Float-based layout** in the header:
  - Club name / logo floated to the left  
  - Navigation menu floated to the right [file:1]
- **Flexbox layout** on the Events page:
  - At least three event cards aligned using Flexbox
  - Each event card includes a title, short description, background color, and border [file:1]
- **Join Club Form** on `join.html`:
  - Name  
  - Branch  
  - Year  
  - Email  
  - Submit button [file:1]

## How to Run

1. Clone or download this repository.  
2. Open `index.html` in any web browser.  
3. Use the navigation bar to visit the About, Events, and Join pages.
