
This repository contains a multi-page College Club website created as part of the Multi-Page College Club web development assignment. The project is built using only HTML and basic CSS, and demonstrates semantic structure, layout techniques, and HTML forms.



The website consists of the following pages: 

- index.html – Home page  
- about.html – About Club page  
- events.html – Events / Activities page  
- join.html – Join Club form page  
- styles.css – Common stylesheet  
  



- Semantic HTML layout using <header>, <nav>, <main>, and <footer> where appropriate. 
- Common Navigation Bar on all pages with links to:
  - Home
  - About
  - Events
  - Join 
- Float-based layout in the header:
  - Club name / logo floated to the left  
  - Navigation menu floated to the right 
- Flexbox layout on the Events page:
  - At least three event cards aligned using Flexbox
  - Each event card includes a title, short description, background color, and border
- Join Club Form on join.html:
  - Name  
  - Branch  
  - Year  
  - Email  
  - Submit button
